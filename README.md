# Yplayer-python-music-app
🎵 A desktop music player built with Python — play, queue, seek, and track your listening stats.
A lightweight desktop music player built entirely in Python using Tkinter and pygame. Supports MP3, WAV, OGG, and FLAC files. Features a clean dark-themed GUI with playback controls, a seekable progress bar, shuffle and repeat modes, and a queue system. All listening history is saved locally to a SQLite database, laying the groundwork for stats and monthly recaps. No internet connection required — everything runs on your machine.
Built with: Python 3 · Tkinter · pygame · mutagen · SQLite
